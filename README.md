# EDA - Análisis exploratorio de clientes

Análisis exploratorio del dataset `clientes.csv` realizado como Actividad N°3 
de la Maestría en Gestión y Desarrollo de IA (CAECE).

## Contenido del repositorio

- `EDA_clientes.ipynb` — Notebook con el análisis completo (ver online en GitHub).
- `EDA_clientes.py` — Script equivalente para ejecutar desde terminal.
- `clientes.csv` — Dataset utilizado.

## Análisis realizado

1. **Inspección inicial** del dataset (estructura, tipos, estadísticas descriptivas).
2. **Filtrado** de clientes de alto valor (ingresos > $50.000) y cálculo de 
   ingresos promedio por estado civil.
3. **Visualizaciones**: distribución de edad y relación entre ingresos y monto 
   de compra.

## Tecnologías

Python · pandas · seaborn · matplotlib
